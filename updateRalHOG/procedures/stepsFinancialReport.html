Upload Financial Report to the Website

  • Please note that prior to doing anything in the production environment you can test in the sandbox environment first. This environment is used for testing and is not seen by members.
    • Your webmaster will also help with any questions you may have.
  1. Login to your account on raleighhog.com
  2. Navigate to the Financial Reports page
  3. Click the Edit tab.
  4. Highlight the report you want to add and click the link icon. It looks like a chain link. See image below
  5. Upload the local PDF to the server as follows
    1. In the window that opens, see image below, click the Browse Server button
    2. If needed, expand the financial_editor subdirectories by clicking the plus sign (+) within the box until you see the year for which you will be updating. Click the directory name, then click Upload from the menu bar. See image below
    3. Click the Choose File button and select the file you want to upload. This file will be on your machine in whichever directory you placed it. When the file finishes loading click the Upload button. Click the x in the upper right to close the window. See image below
    4. The PDF has been successfully uploaded to the server
  6. In the list of files, double click on the file you just uploaded to select it
  7. Click the OK button in the resulting window. See image below
  8. Scroll to the bottom of the page and click the Save button
  9. You're done. The current month's Financial Report is available/viewable by chapter members.
