What to take away

I did not have a good way to share my bookmarks without it looking very clumsy and bulky so I quickly put this page together.

Click links in the dropdowns above to see websites for HOG chapters across the USA

Things to note;

  • HOGScan websites all look similar because the underlying CMS is made for HOG chapters
  • Pinnacle websites all look similar because the underlying CMS is made for HOG chapters
  • Wix websites all look different because each part of the website building platform is chosen by those that put the site together
  • Wordpress websites all look different because each part of the CMS is selected by those that put the site together
  • There is a lot of commonality no matter how the site was built

This is not a proposal to choose one web building site over another. The various chapters show some of the important links that we can consider implementing.
