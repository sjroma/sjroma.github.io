Personalize Your Chapter's Website

Purpose of this Tutorial

This page will show;

  • some common tokens and how to use them on your page(s)
  • how to personalize the chapter rocker on the About Us page
  • how to fix the broken links to the Chapter Charter and Chapter Handbook

Using Tokens

Use tokens to minimize redundant typing and the risk of typos.

The following table shows the token you type in on the page you are editing and the result that is displayed after you Save the page. Note; the display will show your unique chapter information. Raleigh's info is shown for display purposes only. Note2; if you want the octothorpe (number, hash, pound sign) to show in front of your chapter number you need to include it right before the token as shown.

It should be noted that any token used in the calender Body will not be translated when viewing the calendar with the HOG[SCAN] smartphone app.

About Us page - Personalize the Chapter Rocker

By default, the About Us page shows a HOG patch with a generic rocker above it. To have the rocker display your chapter information you will need to edit the page. The following steps will guide you.

  1. Login to your website
  2. Reference the screenshot to see the default rocker showing generic information. To change it to reflect your chapter, click the Edit tab.
    1. Referencing the screenshot below, click the down caret to expand the selection and click HTML code to select it.
    2. Referencing the screenshot below, locate the 9999 in the img (image) tag. Changing only the 9999 to your four digit chapter number. For most, it will be a four digit number. There are a few that will use a six digit number.
    3. The screenshot below shows how I edited the line to use my chapter number
    4. Scroll to the bottom of the page and click the Save button.
  3. Your completed page displays showing the message "Basic page About Us has been updated" and the rocker shows your chapter.

About Us page - Fix the Broken Links

  1. Login to your website
  2. This section will show how to fix the broken Download the Chapter Charter and Download the Chapter Handbook links. Reference the image below to see where they are on your unedited page. Click each one and if you get a "This site can't be reached" page, continue with the following instructions. Note that this bug has been fixed for newly set up websites. It is only an issue with legacy websites.
    1. Referencing the screenshot in step 2 in the previous section, click the Edit tab.
    2. Referencing the screenshot in step 2.1 in the previous section, click the down caret to expand the selection and click HTML code to select it.
    3. Referencing the screenshot below, you will change what is between the quotes (be sure and leave the quotes). Replace https://[hogscan:charter_url] with [hogscan:charter_url] Replace https://https://[hogscan:handbook_url] with [hogscan:handbook_url]
    4. Reference the screenshot below to see the tokens in their proper location. Your edit should look the same. Note that the token contains the https protocol. Do NOT enter it before the token. Your edit should look that same as in the screenshot below.
    5. Click the Save button to finish.
  3. Test each link to verify that they now bring up the Chapter Charter and Chapter Handbook.
