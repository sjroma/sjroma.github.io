cPanel - Setup & Manage Officer Email Accounts

General notes, terminology and policies

  • This environment is different than the website. There is only one, shared, account that is used for logging in. Do NOT change the Username or the Password.
  • This document currently only covers creating an email account and changing an account password. They will show as separate sections within this document;
    • Creating Email Accounts
    • Change an Account Password
  • The login links are for cpanel.raleighhog.com You will need to go to cpanal.yourdomain.topdomain where topdomain is .com, .org or .net

Creating Email Accounts

  1. Login to the shared account for cPanel. As this is shared do NOT change the Username or Password. see image below
  2. After logging in, click Email Accounts in the Email section. see image below
  3. Click the +Create button to start creating the first account. see image below
  4. Referencing the image below
    1. Fill in the Username (for instance, director). Note that your domain name is already populated.
    2. Set the Password. Optionally, click the Generate button to have a random password generated for you.
    3. Click the +Create button to complete creating the account.
    4. Optionally, click the box to "Stay on this page after I click create" if you have additional accounts to create.
  5. Inform each officer of their newly created account and let them know the password. (Also let them know they should change their password)

Change an Account Password

Occasionally, you may have an officer leaving a postion that has forgotten their email account password and it has to be reset so the new officer can access the email. The following steps will walk you through changing an account password.

  1. Login to the shared account for cPanel. As this is shared do NOT change the Username or Password. see image below
  2. After logging in, click Email Accounts in the Email section. see image below
  3. The page opens showing all created email accounts. Click the Manage button for the account whose password is to be reset. The Activies Officer account (activities@raleighhog.com) is shown as an example for this tutorial. see image below
  4. In the page that opens, enter in the New Password. Optionally, you can click on the Generate button to have the system gen a new password for you. When clicked, the eye with the line through it will let you see the password that you are entering.see image below
  5. After entering the new password, scroll down and click the Update Email Settings button to save it. see image below
  6. Inform the user of their new password.
